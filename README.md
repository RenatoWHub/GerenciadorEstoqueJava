Este é um projeto de console desenvolvido para praticar os fundamentos do Java.

O que pratiquei aqui:
1- Arrays Paralelos: Para associar nomes de produtos às suas quantidades.
2- Estruturas de Repetição: Uso de do-while para o menu e for para busca e listagem.
3- Lógica Booleana: Criação de alertas de "Estoque Baixo" e validação de vendas.
4- Tratamento de Dados: Manipulação do buffer do Scanner.
